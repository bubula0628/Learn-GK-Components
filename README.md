##簡介##
EZoApp 所強調的的核心價值：就是「重用、組裝」。任何設計好的 UI 都可以轉化為元件，進一步建立起自己專屬的應用設計 UI 規範與樣式，而 EZoApp 本身也提供了相當豐富的開發元件，這篇介紹將進一步提供給使用者，各個元件基本的使用與操作方式。
