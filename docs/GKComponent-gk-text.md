##gk-text
`gk-text` is heading and text component, and also could be set dynamically.

<br/>
###API

- **.text([val])**：  
  	> Set a new value or get current value. 

    > **val**  
      Type: String  
      Text value

            // getter
            var text = $(selector).gk().text();
            // setter
            $(selector).gk().text('newText');
