##raty
`raty` 讓使用者藉由點選的方式，選擇對應的星星數進行分數評比。

###API
若已由 `isUseGKComponent` 將元件轉換為 GK 元件，則可使用 GK 元件之 API，使用方式就是在元件 id 後方加上 `.gk()`，後方接上 API 名稱即可使用，以下範例使用 id 為 test 的 `raty` 元件。

- **score**：  
  	> 描述。

			$('#test').gk().score();
