##image
`image` 是一個可以在設計畫面中拖拉位置進行排版的圖片元件。  

<br/>
###屬性設定
<table>

<tr>
<th style="background:#ddd;">屬性</th>
<th style="background:#ddd;">設定</th>
</tr>

<tr>
<td>id</td>
<td>元件的 id</td>
</tr>

<tr>
<td>src</td>
<td>圖片的網址</td>
</tr>

<tr>
<td>style</td>
<td>直接編寫元件 DOM inline 樣式</td>
</tr>

</table>

<br/>

`image` 元件沒有提供 api。
