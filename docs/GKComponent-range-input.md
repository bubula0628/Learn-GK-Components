##range-input
`range-input` 是範圍控制桿元件，提供使用者利用控制桿調整範圍數值，該元件拖曳進入設計區域後，預設與標題一起放在 `fieldcontain` 內，也可利用拖拉的方式將其移出 `fieldcontain` 外。

<br/>
