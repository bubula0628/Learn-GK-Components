##youtube
`youtube` 元件讓使用者可直接在行動或平板裝置上瀏覽 youtube 影片  

<br/>
###屬性設定
<table>

<tr>
<th style="background:#ddd;">屬性</th>
<th style="background:#ddd;">設定</th>
</tr>

<tr>
<td>id</td>
<td>元件的 id</td>
</tr>

<tr>
<td>width</td>
<td>影片的寬度</td>
</tr>

<tr>
<td>Height</td>
<td>影片的高度</td>
</tr>

<tr>
<td>src</td>
<td>影片的縮圖網址</td>
</tr>

<tr>
<td>videoId</td>
<td>影片在 youtube 的網址</td>
</tr>

</table>
