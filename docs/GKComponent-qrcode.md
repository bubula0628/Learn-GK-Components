##qrcode
`qrcode` 協助使用者將某段文字內容或網址、電話...等資訊轉換為二維條碼  

<br/>

</tr>

<tr>

<br/>
###API
若已由 `isUseGKComponent` 將元件轉換為 GK 元件，則可使用 GK 元件之 API，使用方式就是在元件 id 後方加上 `.gk()`，後方接上 API 名稱即可使用，以下範例使用 id 為 test 的 `qrcode` 元件。

- **render**：  
  	> 描述。

			$('#test').gk().render();
