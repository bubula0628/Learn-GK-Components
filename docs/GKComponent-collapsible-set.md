##collapsible-set
`collapsible-set` 是點選標題的按鈕之後，進行內容收合或展開的元件，內容包含了一個 `collapsible` 元件，可參考 [collapsible 元件說明](https://github.com/ezoapp/Learn-GK-Components/blob/master/docs/GKComponent-collapsible.md)。
